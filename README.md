
# RCP (Remote Computer Program)

-----

접속된 컴퓨터의 Background 와 실행중인 Visiable Program 을 확인할 수 있는 프로그램.

<br/>

## Idea

<br/>

    온라인 수업시, 게임중인 아이들을 보며 학교와 교육청에서 허가되는 범위의
    감시 프로그램을 만들면 어떨까 하는 생각으로 프로그램을 개발하기 시작했다.
